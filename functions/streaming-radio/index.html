Radio Module

The radio module allows the bot to play an online radio station in your voice channel. Users can start playback, stop it, or view the currently playing station using simple slash commands.

Playing a radio stationâ€‹

To start playing a radio station, use the command below.
The bot must be connected to a voice channel, and the station is selected via autocomplete.

Commandâ€‹

/radio play station:{radio station}

Argumentsâ€‹

ArgDescription
stationThe radio station to play (autocomplete enabled).

Stopping the radioâ€‹

To stop the currently playing radio station and disconnect the bot from the voice channel, use the following command.

Commandâ€‹

/radio stop

Showing the currently playing stationâ€‹

To display information about the radio station that is currently playing, use the command below.

Commandâ€‹

/radio now
