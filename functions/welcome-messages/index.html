Welcome Messages

danger

This feature is currently in the testing phase, so bugs may occur. If you encounter any issues, please report them via our Discord server or using the /feedback command. Thank you!

With the welcome messages feature, the bot can send messages to a selected channel about new users joining or users leaving the server. Below, you will find a guide on how to set up this feature.

Enabling or Disable welcome/leave messagesâ€‹

To enable or disable welcome or leave messages, enter the command below. After entering the command, the bot will send a message to successfully enable or disable welcome/leave messages.

  • Enable join/leave messages
  • Disable join/leave messages
