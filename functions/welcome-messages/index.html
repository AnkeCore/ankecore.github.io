Welcome Messages

With the welcome messages feature, the bot can send messages to a selected channel about new users joining or users leaving the server. Below, you will find a guide on how to set up this feature.

Enabling or Disable welcome/leave messages​

To enable or disable welcome or leave messages, enter the command below. After entering the command, the bot will send a message to successfully enable or disable welcome/leave messages.

  • Enable join/leave messages
  • Disable join/leave messages

Command for welcome messages​

/welcomemessage toggle type:✅ Join Message enabled:True

Command for leave messages​

/welcomemessage toggle type:❌ Leave Message enabled:True

Command for welcome messages​

/welcomemessage toggle type:✅ Join Message enabled:False

Command for leave messages​

/welcomemessage toggle type:❌ Leave Message enabled:False

Setting up welcome or leave messages​

  • Setting up Join messages
  • Setting up Leave messages

Setup command for welcome messages​

/welcomemessage set type:✅ Join Message (The user has joined the server.) channel: {args..}

Setup command for leave messages​

/welcomemessage set type:❌ Leave Message (The user has left the server.) channel: {args..}
warning

Each time the setup command is executed, the previous record will be erased from our memory. Therefore, we recommend saving the command for future updates.

Available arguments in the command:​

ArgDescription
typeMessage type (join/leave)
channelThe channel where the message is sent.
messageThe content of the message may contain placeholders.
embedUse embed message? (true/false)
embed_author_user_avatarUse the user's avatar as the author? (true/false)
embed_author_textText author embed message.
embed_titleTitle embed message.
embed_descriptionDescription of the embed message.
embed_colorEmbed message color (#HEX code).
embed_thumbnails_user_avatarUse a user's avatar as a thumbnail? (true/false)
embed_showtimestampShow timestamp? (true/false)
embed_footer_user_avatarUse the user's avatar as a footer? (true/false)
embed_footer_textText footer embed message.

Available placeholders in the command:​

PlaceholderDescription
{user}Displays user name in format <@373880011536465920>
{userName}Displays user name in format FiloooSK
{serverName}Displays guild name
{membersCount}Displays the current number of members on the server.
{guildId}Displays the server ID
